# arch

Personal Arch Linux setup. Automated install, packages, services, and system/user config for a single workstation (KDE Plasma, AMD GPU).

## Fresh Install

Boot the Arch ISO and run:

```bash
bash <(curl -sL clearcmos.com/go)
```

After reboot, setup runs automatically on first login.

## Re-run Setup

```bash
~/arch/setup.sh
```
